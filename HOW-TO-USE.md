# Fullstack Project Builder - 使用指南

## 这是什么？

这是一个 Claude Skill，让你只需要：
1. **提供高保真 UI 图片**
2. **简单描述功能**

然后 Claude 会：
- ✅ 自动生成完整项目代码
- ✅ 按标准结构组织文件
- ✅ 自动审查代码质量
- ✅ 生成宝塔部署配置

---

## 如何使用

### 第一步：安装 Skill

1. 下载 `fullstack-project-builder.skill` 文件
2. 在 Claude.ai 设置中上传此 Skill
3. Skill 自动生效

### 第二步：提供 UI 和需求

上传你的高保真 UI 图片，然后这样说：

```
请根据这个 UI 帮我生成项目代码。

功能说明：
1. 这是一个 xxx 系统
2. 用户可以 xxx
3. 管理员可以 xxx
4. ...
```

### 第三步：确认分析

Claude 会返回：

```markdown
## UI 分析

**页面识别**
1. 登录页
2. 首页
3. 列表页
...

**组件拆分**
- Button, Input, Card...

**数据结构**
- User: id, name, email...

**API 设计**
- POST /api/v1/auth/login
...

确认以上分析正确吗？
```

你确认后，Claude 开始生成代码。

### 第四步：获取代码

Claude 会输出完整项目，包括：

```
✅ frontend/           - 用户前端
✅ admindashboard/     - 管理后台
✅ backend/            - 后端 API
✅ server/             - WebSocket 服务
✅ database/           - 数据库迁移
✅ docker-compose.yml  - Docker 配置
✅ 代码审查报告
```

### 第五步：部署上线

项目准备上 VPS 时：

1. 在宝塔创建站点
2. 把宝塔默认配置发给 Claude
3. Claude 修改好配置 (反向代理、SSL等)
4. 你粘贴到宝塔 → 完成部署

## 技术栈说明

| 你得到的 | 技术 |
|---------|------|
| 用户前端 | React + TailwindCSS (手写) |
| 管理后台 | Material Kit React |
| 后端 | FastAPI 或 Express (自动选择) |
| 数据库 | PostgreSQL |
| 实时通信 | WebSocket |

**自动包含功能**：
- ✅ 国际化 (中/英切换)
- ✅ 深色/浅色主题
- ✅ 响应式 (手机/平板/桌面)
- ✅ JWT 认证

## 文件说明

当你拿到项目代码后：

```
project/
├── frontend/          # npm install && npm run dev
├── admindashboard/    # npm install && npm run dev
├── backend/           # pip install -r requirements.txt && uvicorn main:app
├── server/            # WebSocket 服务
├── database/          # 数据库迁移文件
├── docs/              # 项目文档
├── uploads/           # 上传文件目录
├── logs/              # 日志目录
├── baota-peizhi/      # 宝塔配置 (上线时用)
└── docker-compose.yml # Docker 启动
```

### 启动开发环境

```bash
# Windows Docker
docker-compose -f docker-compose.dev.yml up
```

访问：
- 前端: http://localhost:3000
- 管理后台: http://localhost:3001
- API: http://localhost:8000
